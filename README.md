# Random Users

A small React + Vite project that fetches and displays a list of random user profiles.

## Features

- Fetches user data from a public API
- Displays user photos, names, and email addresses
- Responsive card layout with hover styling

## Development

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```

## Notes

This project is built with React, Vite, and basic CSS for a clean presentation of random user cards.
